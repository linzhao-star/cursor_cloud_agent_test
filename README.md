# 番茄钟

使用cursor最新推出的增强版cloud agents模式，实测开发本项目：25 分钟倒计时番茄钟网页，使用 React + Vite 构建。

## 功能

- **25 分钟倒计时**：标准番茄钟时长
- **开始**：开始计时（倒计时中时按钮禁用）
- **重置**：停止并重置为 25:00
- **提醒**：当倒计时结束，弹窗提醒

## 运行

```bash
npm install
npm run dev
```

在浏览器打开 http://localhost:5173/ 即可使用。

## 构建

```bash
npm run build
npm run preview  # 预览生产构建
```
